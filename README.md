# ortizgaming-workshops-unittest

Workshop for Unit Tests - Ortiz Gaming

Este workshop será feito utilizando o Microsoft Visual Studio 2019

https://visualstudio.microsoft.com/pt-br/vs/community/ (Versão Free)

A linguagem a ser utilizada é o C# (.NET Core):

https://docs.microsoft.com/pt-br/dotnet/csharp/language-reference/language-specification/introduction

Deverão ser utilizados os seguintes frameworks:

NUnit:

https://nunit.org/
https://www.nuget.org/packages/NUnit/

Moq:

https://github.com/Moq/moq4/wiki/Quickstart
https://www.nuget.org/packages/moq/

Obs.: Os frameworks de teste (Pacotes NuGet) devem ser adicionados APENAS nos projetos de teste.

Crie um fork deste projeto, faça suas alterações e envie um PULL REQUEST para o autor para discussões e ajudas!


Happy Testing! :-D
